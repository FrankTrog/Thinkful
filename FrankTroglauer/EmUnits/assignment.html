Assignment

In this assignment, we will use EMs to size text and other items. In order to measure your understanding of how EMs are calculated, your goal will be to achieve specific pixel dimensions using only EMs. When you're using EMs in the real world, you don't have to worry about how many pixels they will result in (although a lot of people still do). For this assignment, the main goal is to be able to accurately size elements with EMs, and that's why we'll be measuring pixels to evaluate your work.

In the CSS that you write for this assignment, you are only allowed to EM units. In order to see how many pixels they result in, use the developer tools:

  • To see the pixel size of text that you have sized with EMs, select the element and then look for font-size under the "Computed" tab of the "Elements" section in the developer tools.
  • To see the pixel width of an element that you have sized with EMs, select the element and then either look for font-size under the "Computed" tab or look at the illustration of the element under either "Styles" or "Computed".

Note: In order to avoid scaling issues, make sure that the page zoom is set to 100% (the default). Ctrl + 0 achieves this in Google Chrome on Windows.
